## 🔄 Common mkdir Options

| Option | What It Does |
|--------|--------------|
| `mkdir dir` | Create single directory |
| `mkdir dir1 dir2` | Create multiple directories |
| `mkdir -p` | Create parent directories |
| `mkdir -v` | Show verbose output |
| `mkdir -m` | Set custom permissions |

## 🚀 Quick Practice (Copy-Paste Ready)

```bash
# Create single directory
mkdir projects

# Create multiple directories
mkdir logs backups scripts

# Create nested directories
mkdir -p app/config/settings

# Create directory with permissions
mkdir -m 755 secure_folder

# Show verbose output
mkdir -v testdir
```
